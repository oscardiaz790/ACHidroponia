 HIDROVERDE - PÁGINA WEB DE TU NEGOCIO
==============================================

CÓMO ABRIR EN VISUAL STUDIO CODE:
1. Descomprime el archivo ZIP
2. Abre VS Code → Archivo → Abrir Carpeta → selecciona "hidroponica"
3. Instala la extensión "Live Server" (botón de extensiones, buscar Live Server)
4. Haz clic derecho en index.html → "Open with Live Server"

ESTRUCTURA DE ARCHIVOS:
  index.html        → Página principal (edita aquí el contenido)
  css/styles.css    → Estilos y colores
  js/main.js        → Animaciones y formulario
  img/              → Carpeta para tus fotos (agrégalas aquí)
